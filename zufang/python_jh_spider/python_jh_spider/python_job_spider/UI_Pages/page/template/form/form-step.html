说明


一、面向用户

全体高校毕业生


二、设置方式

方法一：登录个人账号->进入我的世界->密码修改模块->填写信息->确认信息->密码修改完成

方法二：登录个人账号->点击右上角个人头像->密码修改弹窗->填写信息->确认信息->密码修改完成


三、注意事项

1.提醒短信将会发送到您的绑定手机上；

2.提醒短信仅提醒您密码进行了修改，但不会告知修改后的新密码；

3.当出现您的帐号被异常修改密码，且您是二代密保且设置密保手机的用户，将收到由安全中心向您的密保手机下发的提醒短信；

4.提醒短信不扣除短信提醒条数；

5.预付费、年费用户需绑定移动、联通、电信手机才可以使用。
